# Keyword Mapping for SEO

Source: https://www.semrush.com/blog/keyword-mapping/
Author: Rachel Handley (Senior Content Writer), contributors Chris Shirlow and Simon Fogg
Published/updated: 2024-12-12

## Definition
Keyword mapping assigns target keywords to specific website pages, tracked in a keyword map document, matching each keyword to the URL that best satisfies search intent.

## Why It Matters
- Prevents keyword cannibalization
- Improves search visibility through better content organization
- Identifies content gaps requiring new pages
- Maximizes total keywords a site can rank for
- Saves long-term time by preventing future re-optimization

## Step-by-Step Process
1. **Download Template** — free Google Sheets keyword mapping template.
2. **Identify Topic Areas** — define broad subject categories based on niche/products/services (example: dog training → puppy training, dog behaviors, agility training).
3. **Group Keywords into Clusters** — use the Keyword Strategy Builder to organize into pillar pages, subpages, and keyword clusters.
4. **Select Relevant Ideas** — review Search Intent, Volume, Keyword Difficulty (KD%); export selected pages/keywords to spreadsheet.
5. **Assign URLs** — match target URLs to pillar/subpages by search intent; mark pages "To optimize" or "To create" in a Status column.

## Implementation Recommendations
- **Optimize Existing Pages**: filter "To optimize," use On Page SEO Checker, place primary keywords in title/meta/H1/body, secondary keywords naturally in body/subheadings, update Status/Last Updated columns.
- **Create New Content**: filter "To create," prioritize by volume/difficulty, use SEO Content Template (semantically related keywords, text length/readability, link building, competitive analysis).
- **Optimize Internal Linking**: use keyword-rich anchor text, reference the keyword map when linking, follow Google's link spam guidelines for backlinks.
