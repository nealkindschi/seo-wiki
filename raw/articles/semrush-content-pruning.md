# Content Pruning

Source: https://www.semrush.com/blog/content-pruning/
Author: Sydney Go
Published/updated: 2024-05-09

## Definition
Removing underperforming or outdated content to enhance SEO results, via three actions: refresh (improve quality/accuracy), consolidate (merge similar pages), or remove (delete entirely).

## Why Prune
- Improved rankings by eliminating outdated/inaccurate information
- Enhanced UX through current, relevant content
- Increased organic traffic by preventing cannibalization and duplicate-content issues
- More efficient crawl budget (critical at 10,000+ pages)

Case study: Semrush's own blog saw a "substantial increase in organic traffic" through multi-year content refreshing efforts.

## When to Prune
- Large sites: every 1-3 months (batch processing)
- Small sites: once or twice yearly

## 7-Step Process
1. **Define project scope** — determine which page types to audit; use Site Audit for a full page inventory.
2. **Identify low-performing content** — set a performance threshold (example: minimum 1,000 monthly organic sessions); track via Google Analytics; log in a spreadsheet.
3. **Identify issues** — review technical problems via Site Audit; for pages without technical issues, evaluate against search intent and competitors.
4. **Review backlink data** — use Backlinks tool; pages with substantial backlinks should avoid deletion (SEO damage risk).
5. **Label content** — technical issue / outdated / thin content / intent mismatch / duplicate content / needs backlinks.
6. **Determine next steps** — assign refresh, consolidate, or remove.
7. **Release changes in stages** — deploy incrementally, measure impact before continuing.

## Decision Criteria
- **Refresh**: intent mismatch, outdated, thin content, technical issues — expand content, update info, resolve issues, keeps backlink equity.
- **Consolidate**: duplicate content — 301-redirect to the highest-quality version, transfers link equity.
- **Remove**: outdated/irrelevant pages with no link value — creates 404s, causes lost organic traffic, breaks internal links (use cautiously).

## Tools
Semrush Site Audit, Organic Traffic Insights, Backlinks; Google Analytics, Google Search Console.
