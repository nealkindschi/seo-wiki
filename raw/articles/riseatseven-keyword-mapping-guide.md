# The Beginner's Guide to Keyword Mapping

Source: https://riseatseven.com/blog/keyword-mapping-guide/
Author: Lidia Infante (Rise at Seven)

## Core Definition
Keyword mapping is the process of pairing target keywords to their goal landing pages, aiming to rank the right pages for the right (conversion-focused) keywords.

## Key Benefits
1. Supports an informed internal linking strategy
2. Measures SEO effectiveness with precision
3. Creates more persuasive reporting and stakeholder communication
4. Enables rapid response to performance declines

## Step-by-Step Process

### Step 1: Identify Top Pages
- Access Google Analytics (Behavior → Site Content → Landing Pages)
- Use a 12-month date range to minimize seasonal variations
- Filter for organic traffic from target countries
- Export data showing page, sessions, conversions, conversion rate
- Remove non-business-critical pages (testimonials, policy pages, irrelevant blog content)
- Select approximately 20 priority pages for mapping

### Step 2: Establish Seed Keywords
- Primary keywords typically appear in page URLs
- Validate selections using Google Search Console top queries
- Cross-reference with keyword research tools (comparing volume, difficulty, intent)
- Consult PPC data for additional validation
- Balance volume, difficulty, and conversion potential

### Step 3: Expand Keyword Variations
- Mine Google Ads search term reports
- Extract ranking keywords from Google Search Console
- Use Ahrefs and SEMrush keyword clustering features
- Narrow findings to approximately 10 keyword variations per page
- Analyze SERPs for user intent and competitive presence
- Note if pages feature heavy advertising (indicates commercial value)

### Step 4: Build the Keyword Map
- Organize data using a Google Sheets template
- Import refined keyword groups into rank tracking software
- Tag keywords by main keyword group for performance monitoring
- Monitor for cannibalization issues (multiple pages targeting identical queries)

### Step 5: Create an On-Page Optimization Roadmap
- **Missing Content**: create new pages for important keywords lacking dedicated content
- **Non-Ranking Content**: troubleshoot via technical audits (robots.txt, noindex, HTTP response codes, page experience), content relevancy review, internal linking opportunities, competitive content benchmarking
- **Keyword Cannibalization**: either consolidate pages or reoptimize one page for long-tail variations

## Recommended Tools
- Rank Trackers: SEMrush, Ahrefs, SEOmonitor
- Analytics: Google Analytics, Google Search Console
- Keyword Research: Ahrefs, SEMrush
- Audit: Screaming Frog

## Recommendations
- Tailor the framework to site maturity and existing SEO efforts
- Use market knowledge alongside data
- Focus on bottom-of-funnel keywords delivering business value
- Monitor keyword group performance trends, not individual keywords in isolation
