Rachel Handley et al., "Keyword Mapping for SEO" (Semrush blog, published/updated 2024-12-12).

## Key takeaways

- Same core practice as [[riseatseven-keyword-mapping-guide]] — assign
  target keywords to specific pages, track in a keyword map document —
  but starts from **topic areas** rather than existing GA traffic:
  define broad subject categories → cluster keywords under
  pillar-page/subpage structure (via Semrush's Keyword Strategy Builder)
  → filter by search intent/volume/KD% → assign URLs and mark each
  page's status as "To optimize" or "To create."
- Reinforces (agrees, doesn't conflict with) Rise at Seven's
  missing-content / non-ranking-content / cannibalization framing: same
  three outcomes, described here as "To create," "To optimize," and
  cannibalization prevention.
- Adds concrete on-page tactics once a page is in the map: primary
  keyword in title/meta description/H1/body, secondary keywords
  naturally in body/subheadings; and an internal-linking tactic —
  reference the keyword map when choosing anchor text so links stay
  keyword-rich and non-cannibalizing (ties to
  [[link-and-anchor-text-best-practices]]'s anchor-text guidance).
- Tools: Keyword Strategy Builder (AI clustering), On Page SEO Checker,
  SEO Content Template (semantic keywords, length/readability
  benchmarks, competitive analysis), SEO Writing Assistant, Backlink
  Audit.
