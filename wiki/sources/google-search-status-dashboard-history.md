# Google Search Status Dashboard — History

**Citation:** Google. "Search Status Dashboard — History." Live page,
snapshot fetched 2026-07-07.
https://status.search.google.com/products/rGHU1u87FJnkP6W2GwMi/history

**Note**: this is a live, continuously-updated status page, not a fixed
document — Google appends new entries as new ranking updates/incidents
occur. Re-fetch periodically to keep the wiki's timeline current; this
ingest is a snapshot, not a closed record.

## Key takeaways

- Lists ~34 dated, resolved ranking updates and incidents from July 2021
  through June 2026, each with an exact start date and duration.
- Update types: core updates, spam updates, link spam updates, helpful
  content updates, product/reviews updates, one page experience update,
  one Discover update, and two "ranking ongoing issue" incident reports.
- **Helpful content updates** appear as a distinct named type from
  August 2022 to September 2023 (3 instances) and never again —
  consistent with this system being folded into core ranking rather than
  shipped as a separate named update afterward.
- **Product/reviews updates** appear from December 2021 to November 2023
  (7 instances) and never again — same pattern.
- Core updates are the most frequent type, recurring roughly every 2-4
  months throughout the whole period.
